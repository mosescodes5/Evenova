02 · Features

Everything in one platform

Evenova replaces five separate tools with one cohesive system built specifically for the Nigerian event context.

🎟

Multi-Tier Ticketing

Create unlimited ticket tiers (Early Bird, Regular, VIP, VVIP) with custom pricing, quantities, colors, and perks listed on the ticket.

📱

QR Code Scanning

Gate staff scan tickets using any smartphone camera or dedicated PDA scanner. Works fully offline and syncs when reconnected.

🔐

Cryptographic Tickets

Every ticket is cryptographically signed. Duplicates and fakes are detected instantly at the gate, eliminating fraud entirely.

🚪

Multi-Gate Management

Define multiple entry gates (Main, VIP, Press, etc.) with unique colors. Each gate only admits the ticket types assigned to it.

📊

Live Dashboard

Real-time check-in counter, capacity meters, revenue tickers, and gate-by-gate activity — all updating live during the event.

📧

Email Blast Engine

Upload an Excel list, AI-clean messy data, compose rich HTML emails, and blast thousands of attendees via Resend, Brevo, or AWS SES.

💰

Revenue Analytics

Per-event and platform-wide revenue breakdown by ticket tier. Track check-in rates, revenue per organizer, and scan success rates.

👥

Team Management

Organizers create staff accounts with role-based access. Assign scanners to specific gates. Remove access instantly after the event.

📝

Custom Registration Forms

Add custom fields to any event — T-shirt size, dietary requirement, company name, job title. All captured at ticket purchase.

💳

Payment Integration

Connect Paystack or Flutterwave for instant online payment. Bank transfer mode for offline payers with manual ticket issuance.

📋

Scan Log & Audit Trail

Every scan attempt is logged with timestamp, gate, staff member, and outcome. Full audit trail for post-event reporting.

🛡

Admin Control Panel

Platform-level admin reviews organizer applications, approves accounts, monitors all events, and oversees platform-wide revenue.

📶

Offline Mode

Gate scanners can download a local ticket cache before the event. Scanning continues even when internet drops at the venue.

🤖

AI Email Cleaning

Upload any messy Excel file and Claude AI extracts, deduplicates, and fixes email addresses automatically — no manual cleanup needed.

🔔

Email Verification

New organizers receive a 6-digit verification code by email. Account auto-activates on confirmation — no admin bottleneck.

☁️

Cloud Database

Powered by Supabase (PostgreSQL). All data persists in the cloud — no data loss, accessible from any device, anywhere.

03 · User Roles

Three levels of access

Evenova's role-based system ensures each user sees only what's relevant to their job.

Role Who Access
Platform Admin Evenova owner Full platform view — approve/reject organizers, monitor all events, platform revenue, email blast to all, scan logs across all events
Organizer Event company or individual Create and manage their own events, issue tickets, manage their team, view their revenue, send email blasts to their attendees
Staff / Scanner Gate staff, coordinators QR scanner only — assigned to specific gates. Cannot access events, revenue, or settings. Credentials expire on removal.

🔐 Admin Access

The platform admin account is accessed via a hidden keyboard shortcut (Ctrl+Shift+A) on the login screen — invisible to regular users. This keeps the admin interface completely separate from the organizer-facing product.

04 · Ticketing System

Cryptographically signed tickets

Every Evenova ticket carries a unique cryptographic signature. It cannot be duplicated, screenshotted, or transferred fraudulently.

1

Organizer creates event

Define ticket tiers (name, price, quantity, perks, color), entry gates, and a custom registration form with any fields needed.

2

Attendee registers & pays

Public event page collects registration data, processes payment via Paystack or Flutterwave, and instantly generates a signed QR ticket.

3

Ticket delivered by email

A branded HTML email containing the QR code, event details, ticket tier, and holder name is sent immediately to the attendee's inbox.

4

Attendee arrives at the gate

Attendee shows the QR code on their phone. The gate scanner reads it in under 1 second.

5

System verifies & admits

The system checks: valid signature, correct gate, not already scanned, correct ticket tier. Outcome displayed instantly — green for admitted, red for rejected.

Scan OutcomeMeaningAction
✅ AdmittedValid ticket, first scan, correct gateLet in
❌ RejectedInvalid QR or tampered codeDeny entry
⚠️ DuplicateTicket already scannedDeny — possible transfer attempt
🚪 Wrong GateTicket valid but wrong entranceRedirect to correct gate
05 · Gate Scanning & Hardware

Works on any device — including PDAs

Evenova's scanner is a web app that runs on any device with a camera and browser. No app installation needed — staff just open a link.

📱

Smartphones

Any Android or iPhone with a browser. Staff log in, select their gate, and start scanning. Works on cheap ₦30,000 phones.

🖥

PDAs / Barcode Scanners

Dedicated PDA devices (Zebra, Honeywell, Urovo, etc.) with built-in QR scanners connect via browser. Faster, more durable, and ideal for high-volume gates.

📶

Offline Mode

Download a local ticket cache before the event. Scanning works even if venue WiFi fails. Logs sync automatically when connection returns.

⚡

Sub-second Verification

QR decode and cryptographic verification happens locally in under 500ms. No server round-trip needed at the gate.

💡 PDA Recommendation for Sponsors

For professional event deployments, we recommend:

  • Urovo DT50 — rugged Android PDA with integrated 2D QR scanner, widely available in Nigeria (~₦120,000)
  • Zebra TC21 — enterprise-grade, excellent for outdoor events (~₦180,000)
  • Any Android tablet + external Bluetooth scanner — most affordable option (~₦60,000 total)

All three work with Evenova out of the box via Chrome browser.

06 · Email Blast Engine

Mass email built for event marketing

Evenova's built-in email blast tool lets organizers and the platform admin send thousands of personalised emails without any third-party marketing tool.

📊

Excel Upload

Upload any .xlsx or .csv file. Evenova auto-detects email columns and extracts addresses — even from messy, inconsistently formatted files.

🤖

AI Cleaning

One click sends raw Excel rows to Claude AI, which extracts, deduplicates, fixes typos (gmai.com → gmail.com), and returns a clean list.

🎨

Rich HTML Composer

Write emails in Rich, HTML, or Plain text mode. Use {name} tags for personalisation — each recipient gets their name in the greeting.

📬

Multiple Providers

Send via Resend.com (simplest), Brevo, or AWS SES. Free tiers available on all three. Switch providers without changing email templates.

👥

Import Attendees

One click imports all registered attendees from any event directly into the recipient list — perfect for event follow-ups and announcements.

📈

Send History

Every blast is logged with subject, recipients, sent count, failed count, provider used, and a preview snippet for reference.
